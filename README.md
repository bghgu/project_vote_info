### 실행

 - `nodejs` 와 `npm` 을 설치합니다. 설치 방법은 [nodejs.org](https://nodejs.org) 를 참고하세요.
 - 실행에 필요한 의존성을 설치합니다.

  ```bash
  npm install
  ```
 - node.js 코드 실행를 실행합니다.

  ```bash
  npm start
  ```
  - `localhost:3000`으로 실행이 가능합니다
  - 중지하려면, 키보드에서 `Crtl + C`를 누릅니다.
