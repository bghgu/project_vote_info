# 선거 실시 여부 확인(`/check`)
---
## 학생의 선거 실시 여부 확인

메소드 | 경로 | 짧은 설명
--- | --- | ---
POST | / | 선거 실시 여부 확인

### 요청 헤더
~~~
Content-Type: application/json
~~~

### 요청 바디
#### 설명
```json
{
    "id": "학생의 학번"
}
```
#### 예
```json
{
    "id": "201232016"
}
```
### 응답 바디
#### 투표 실시
```json
{
    "message": "투표를 했습니다."
}
 ```
#### 투표 미실시
 ```json
 {
     "message": "아직 투표를 하지 않았습니다. 당신의 후보에게 투표하세요."
 }
 ```
#### 학생 정보 없음
 ```json
 {
     "message": "등록된 학생이 아닙니다."
 }
 ```
