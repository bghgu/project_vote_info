# 선거권 확인(`/suffrage`)
---
## 학생의 선거권 확인

메소드 | 경로 | 짧은 설명
--- | --- | ---
POST | / | 학생의 선거권 확인

### 요청 헤더
~~~
Content-Type: application/json
~~~

### 요청 바디
#### 설명
```json
{
    "id": "학생의 학번"
}
```
#### 예
```json
{
    "id": "201232016"
}
```
### 응답 바디
#### 선거권 확인
```json
{
    "message": "선거권이 있습니다."
}
 ```
#### 선거권 없음
 ```json
 {
     "message": "선거권이 없습니다."
 }
 ```
#### 학생 정보 없음
 ```json
 {
     "message": "등록된 학생이 아닙니다."
 }
 ```
